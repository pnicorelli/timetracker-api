What?

This project is a MVP for a timetracker writed in JS.

The timetracker works with 3 different entity:

  • API SERVER
  • WEB FRONTEND for employers
  • MOBILE APP for employees

API SERVER

This is the functionality core. Writed in nodejs and mongodb it serve the frontend and the mobile. It scalable and can be deployed on *openshift* infrastructure in a minute (you just need to drop your custom config on `~/app-root/data/config/production.json` in the instance).

WEB FRONTEND

This is the tool where the employer create new user for the app. Writed in angularjs it can be easly dropped on *surge*.

MOBILE APP

This is the app used by the employees. Writed in angularjs and ionic, can be compiled for android and ios.
